## Slice Schema Definition
Here's definition for how to the slices schema is built

- A course has many lessons.
- A lesson belongs to one course.
- A user can enroll in many courses.
- A course can have many enrolled users.
- A user can complete many lessons.
- A lesson can be completed by many users.
